<br>

## <picture><img src="https://github.com/7oSkaaa/7oSkaaa/blob/main/Images/Statistics.gif?raw=true" width=50px></picture> Research focus

Main topics that motivate the repositories in this organization:

- Intelligent control for robotic systems, manipulators, and mobile robots.
- Neural network based controllers and learning based control strategies.
- Wavelet analysis for signal processing, system identification, and fault detection.
- Robust and adaptive control for systems with uncertainty.
- Simulation and numerical experiments for validation.

<br>

## <picture><img src="https://github.com/7oSkaaa/7oSkaaa/blob/main/Images/Right_Side.gif?raw=true" width=50px></picture> Repositories you will find here

Typical types of repositories:

- Core libraries for control, simulation, and identification.
- Experimental code for PhD and MSc research.
- Reproducible material for papers, theses, and talks.

You can check the description of each repository to see its status, scope, and intended use.
